------README------------------------------------------------------------------
------------------------------------------------------------------------------
Extras folder is for add-ons to QTI for easy compatability with well-known packages/tools.
------------------------------------------------------------------------------
1. FmodIntegration:

Explanation:------------------------------------------------------------------
	Comes with PlayOneShot and PlayEvent interactions that use the Fmod API, 
	if Fmod is present in your project you can use these interactions instead of the default ones that use the unity sound system. 
	If Fmod is not present they won't be usable.

How to use:-------------------------------------------------------------------
	Just double click the package and import the incoming scripts, then, 
	these interactions will start showing up in the interactions dropdown, it's that easy!

2. QTI_Demos_URP17:

Explanation:------------------------------------------------------------------
	Updates demo scenes shaders to URP. Compatible with URP17.

How to use:-------------------------------------------------------------------
	Just double click the package and import the incoming scripts, then, 
	the demo scene shaders will be URP compatible, it's that easy!
